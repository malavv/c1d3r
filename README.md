# c1d3r
An interactive cider brewing guide.

## Prerequisite
* NPM
  Get from their website.
* Bower
  npm install -g bower
* gulp-cli
  npm install -g gulp-cli
* gulp web component tester
  npm install -g Polymer/web-component-tester
* gulp vulcanize
  npm install -g vulcanize

## Build and serve.
gulp serve:dev

## Test
gulp test:local
